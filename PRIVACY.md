# Privacy Policy

Docs-to-Notion Structurer is a Google Workspace add-on that converts Google Docs content into structured Notion pages.

- **Data Access**: The add-on reads Google Docs content (text, headings, lists) for conversion. No file content is stored permanently.
- **Data Storage**: Notion API keys are stored in Script Properties. No Docs content is stored permanently.
- **Data Sharing**: Docs content is sent to Notion via API to create pages. This is the intended functionality.
- **External API**: The add-on uses the Notion REST API to create pages. Requires user-provided API key.
- **Logging**: Operational logs are written to Apps Script Logger for troubleshooting.
- **Removal**: Remove API keys and settings by uninstalling the add-on or clearing Script Properties.
- **Support**: For privacy questions, open an issue in the repository or contact support@tangentforge.com.
